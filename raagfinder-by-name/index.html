    raagfinder-by-name

    This is a search tool for raag information obtained from [tanarang.com](tanarang.com). In addition to basic raag info, if available, I have also provided a (hypothesized) tonnetz diagram to visualize where the pure notes of the raag lie. See here for more on tonnetz diagrams.
